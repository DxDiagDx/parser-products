# Инструкция для AI-агента

Ты помогаешь создавать парсеры интернет-магазинов на основе шаблона parser-products.

## Контекст

Этот шаблон предназначен для парсинга товаров в три этапа:
1. **Сбор ссылок** — пагинация, получение url товаров и цен
2. **Сбор товаров** — описание, характеристики, изображения
3. **Скачивание фото** — сохранение изображений

## Структура базы данных

- **products** — товары: url (уникальный), цены, stock_status, parse_status
- **params** — характеристики: key-value привязка к товару
- **images** — изображения: url, position (порядок), local_name (путь к файлу)

## Обязательные правила

1. Все сетевые запросы делай через `requests_session.create_session()` и `get_request()`
2. При сохранении товара используй `insert_product()` из `utils/queries.py` — он автоматически обновляет существующие
3. Изображения сохраняй с position по порядку обнаружения на странице (0, 1, 2...)
4. Параметры сохраняй пачкой через `insert_params()`
5. Для каждого магазина адаптируй селекторы в примерах, но не меняй структуру БД
6. Добавляй обработку ошибок — если нет цены, ставь None; если нет фото, пропускай
7. Используй loguru для логирования

## Типовые задачи

### Создать парсер для нового магазина
1. Скопируй `examples/01_get_links.py` и адаптируй селекторы под пагинацию магазина
2. Скопируй `examples/02_get_details.py` и адаптируй селекторы для карточки товара
3. Проверь, что prices и old_prices корректно извлекаются
4. Запусти этапы последовательно

### Обновить существующий парсер
- Если изменилась структура страницы — обнови селекторы
- Если нужно добавить новое поле в products — сначала измени `schema.sql`, затем `queries.py`

## Частые проблемы

- **Нет цены в пагинации** — ставь None, цена будет из детальной страницы
- **Дублирующиеся url изображений** — `download_images.py` обрабатывает это автоматически
- **Сайт блокирует** — используй прокси из config.py
- **JS-подгрузка** — этот шаблон не использует Selenium. Используй браузерное расширение или другой способ для получения готового JSON
